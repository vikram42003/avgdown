# avgdown

A Turborepo-powered monorepo.

## Structure

- `apps/*`: Applications
- `packages/eslint-config`: Shared ESLint configurations
- `packages/typescript-config`: Shared TypeScript configurations

## Getting Started

### Prerequisites

- [Bun](https://bun.sh) (v1.3.10 or higher)
- [Turbo](https://turbo.build)

### Installation

```sh
bun install
```

### Development

```sh
bun run dev
```

### Build

```sh
bun run build
```

### Linting

```sh
bun run lint
```

### Type Checking

```sh
bun run check-types
```
